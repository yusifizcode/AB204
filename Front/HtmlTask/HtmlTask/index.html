About this page

This is a great place to talk about your webpage. This template ispurposefully
unstyled so you can use it as a boilerplate or starting point for you own landing
page designs!This template features:

  • Clickable nav links that smooth scroll to page sections
  • Responsive behavior when clicking nav links perfect for a one page websites
  • Bootstrap's scrollspy feature which highlights which section of the page you're on in the navbars
  • Minimal custom CSS so you are free to explore your own unique design options
